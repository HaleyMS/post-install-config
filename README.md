<h1>osTicket - Post-Install Configuration</h1>
This page outlines the post-install configuration of my open-source help desk ticketing system osTicket.<br />


<h2>Video Demonstration</h2>

- ### [YouTube: How To Configure osTicket, post-installation](https://www.youtube.com)

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)

<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)

<h2>Post-Install Configuration Objectives</h2>

- Item 1: Configuring Roles
- Item 2: Configuring Teams
- Item 3: Configuring Agents and Clients 
- Item 4: Configuring SLA
- Item 5: Configuring Help Topics
